The software architectures used in this repository are:
1. Behavior Models written with SCADE
2. Architecture Models written with AADL

Tools used for computing complexity metrics include:
1. SCADE plug-in tool (scade-metrics) for SCADE models
2. OSATE plug-in tool (aadl-complexity) for Architecture Models written with AADL

The tools are released under an open-source license.
