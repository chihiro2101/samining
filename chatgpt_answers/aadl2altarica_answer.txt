The software architecture used in this repository includes:
1. AADL (Architecture Analysis and Design Language) - used as the input model for the translator.
2. EMV2 (Error Model Annex version 2) - subclauses used in the AADL instance model.
3. AltaRica (version 3) - the output model generated by the translator.
4. OpenAltaRica - the tool that can import the AltaRica model for further analysis.
