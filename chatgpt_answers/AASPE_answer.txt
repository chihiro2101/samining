The software architecture used in this repository includes:
1. AADL (Architecture Analysis and Design Language)
2. OSATE toolset

The tools in this repository are designed for making secure and safe safety-critical systems and provide features such as Attack Impact meta-model, Attack Tree editor, Bridge from Attack Impact to Attack Tree, and Bridge from AADL to Attack Impact. The tools can be installed using the OSATE toolset and the installation process is documented in the README file.

Examples to exercise the tools are available in the `edu.cmu.aaspe.examples` project. The documentation for the tools can be accessed within the tool (OSATE help) or by browsing the related documentation project on GitHub.
