The software architecture used in this repository is based on the MDCF Architect, which is a plugin for Osate 2. It translates a selected subset of AADL into code runnable on the MDCF platform. The architecture involves integrating with Osate 2 for AADL translation and utilizes Maven for building the translator. It also mentions the need for Java (version 8+) to be installed when using Eclipse for development. The project is publicly available under the EPL license. Additionally, the project mentions the use of icons from famfamfam's silk icon set and CSS from Marked.app.
