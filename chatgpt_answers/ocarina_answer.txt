The software architectures used in this repository include:
1. Front-end supporting AADLv2 language, the Behavioral annex, and ARINC653 and EMV2 annexes.
2. Back-end supporting code generation towards AADL runtime, WCET analysis, scheduling analysis, model checking, and constraint analysis.
3. Integration with the AADLib library of AADL components.
4. Embedding in AADL editors such as OSATE using the OSATE2 plugin, AADL Inspector, and TASTE by the European Space Agency.
