The software architectures used in this repository are:

1. AADLv2: AADLib is a library of reusable AADLv2 components.
2. REAL: AADLib components are associated with property sets and validation theorems based on REAL. 

Additionally, the repository mentions integration with the following tools:
1. Ocarina: AADLib is recommended to be installed in conjunction with Ocarina.
2. OSATE: AADLib can be imported as an AADL project in OSATE.
3. AADL-Inspector: AADLib examples can be imported into AADL-Inspector through the GitHub import facility.
