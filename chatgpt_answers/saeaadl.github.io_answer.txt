Software Architectures:
1. Static website architecture using Jekyll: AADL.info website is built using the Jekyll static website builder. Jekyll is a simple, blog-aware, static site generator that allows for easy customization and organization of website content.
2. GitHub Pages architecture: The website is hosted on GitHub Pages, which is a static site hosting service that is directly integrated with GitHub repositories. This architecture allows for seamless integration with the version control system and easy deployment of the website.
